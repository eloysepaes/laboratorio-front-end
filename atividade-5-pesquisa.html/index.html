O elemento HTML < sub > define o texto subscrito.

O texto subscrito pode ser usado para fórmulas químicas, como CO2 ou para escrever palavras em baixo

Exemplo simples de uso

Um exemplo simplesé: O elemento CO < sub > 2 < /sub > que é dióxido de carbono. Neste exemplo o 2 será exibido como subscrito abaixo da linha de base da palavra "CO"
